## 🎬 ¿Qué es un Walk Cycle?

Un **walk cycle** es una secuencia de fotogramas que crea la ilusión de movimiento continuo de un personaje caminando. Se compone de **4 fases principales**:

1. **Contacto** - El talón toca el suelo
2. **Descenso** - El peso se transfiere a la pierna de apoyo
3. **Paso** - Las piernas se cruzan
4. **Elevación** - Preparación para el siguiente contacto

Este ciclo se repite sin interrupciones, haciendo que parezca que el personaje camina indefinidamente.

## 🤖 Uso de IA en el Proyecto

### IA como Herramienta de Apoyo
- **Generación de referencias** - Crear variaciones visuales de poses
- **Brainstorming** - Explorar diferentes enfoques
- **Verificación técnica** - Comprobar anatomía y coherencia
- **Aceleración del proceso** - Ahorrar tiempo en tareas repetitivas

### Importancia de la Creatividad Personal
⚠️ **Recuerda**: La IA es un asistente, **no un sustituto creativo**

- Tu **visión artística** es lo más valioso
- Usa IA para **explorar**, no para copiar
- **Verifica y cuestiona** lo que la IA genera
- Mantén tu **autenticidad** en cada decisión
